### 核心解释

> 源码参考：https://codepen.io/soufiane-khalfaoui-hassani/pen/LYpPWda

<br>

其余的都容易理解，这里关键看 username 和 password 编辑框在点击后，对应的文字变小上移的动效实现

看源码表单设计，非常简单的两个 input 以及两个 label

```html
<div class="login-box">
  <h2>Login</h2>
  <form>
    <div class="user-box">
      <input type="text" name="" required="" />
      <label>Username</label>
    </div>
    <div class="user-box">
      <input type="password" name="" required="" />
      <label>Password</label>
    </div>
    ...
  </form>
</div>
```

<br>

关键 CSS 代码解读：

```css
/* 1. 将二者变成相对布局 */
.login-box .user-box {
  position: relative;
}

/* 2.配置输入框：border以及outline设置可以隐藏默认UI */
.login-box .user-box input {
  width: 100%;
  padding: 10px 0;
  font-size: 16px;
  color: #fff;
  margin-bottom: 30px;
  border: none;
  border-bottom: 1px solid #fff;
  outline: none;
  background: transparent;
}
/* 3.label使用绝对定位，直接贴在input左侧，使得input和label重合 */
/* 可以看见这里设计了transition，代表label将要指向的动画 */
.login-box .user-box label {
  position: absolute;
  top: 0;
  left: 0;
  padding: 10px 0;
  font-size: 16px;
  color: #fff;
  pointer-events: none;
  transition: 0.5s;
}

/* 
4.重点来了
当编辑框input获得焦点（input:focus）或者内容非空（input:valid）时，对应label应当使用的变换  
top为负数表示上移，left左顶格，改变fontsize使得文字变小，顺便这里还改了color
*/
.login-box .user-box input:focus ~ label,
.login-box .user-box input:valid ~ label {
  top: -20px;
  left: 0;
  color: #03e9f4;
  font-size: 12px;
}
```
